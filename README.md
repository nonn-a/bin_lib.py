# Bin_lib.py

A small python library for binary numbers.

## Getting Started

### Dependencies

* Python 3.

### Installing

* Download the file `bin_lib.py`
* Place it in the same folder of your main code.
* Import with `import "bin_lib.py"`.
* Enjoy.
